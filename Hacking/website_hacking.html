How to attack a website

Methods

  • An application installed on a computer - web app penatration
  • Computer uses an OS + other apps - server side attacks
  • Managed by humans - client side attacks

Information gathering

Details

  • Ip address
  • Domain name info
  • Technologies used
  • Other website on the same server
  • DNS records
  • Unlisted files, sub domains, directories
  • Whois Lookup - Find info about the owner of the target
  • Netcraft Site Report - Shows technologies used on the target
  • Robtex DNS Lookup - Shows comprehensive info about the target website
  • Using bing.com, Search for ip:[target ip] - get all website that hosted in same server

    To discover all sub domains
    # knoxkpy google.com

    To discover folders based on given wordlist
    # dirb http://1.1.1.1
