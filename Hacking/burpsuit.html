Burp Suite

About

Burp or Burp Suite is a set of tools used for penetration testing of web applications. It is developed by the company named Portswigger, which is also the alias of its founder Dafydd Stuttard. BurpSuite aims to be an all in one set of tools and its capabilities can be enhanced by installing add-ons that are called BApps.

Configuration

  • Download and Install burpsuite community in KALi (Pre insatlled)
  • Open Burp Suite
  • Open FireFox browser goto settings , search for proxy then set manual proxy URL(127.0.0.1) P(8080)
  • In browser type http://burf download certificates
  • Again goto settings in browser search certificates , then goto authorities tab then import downloaded certificates.
